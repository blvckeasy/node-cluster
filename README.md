## Run Locally

Clone the project

```bash
  git clone https://github.com/blvckeasy/node-cluster.git
```

Go to the project directory

```bash
  cd node-cluster
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

... or start for development

```bash
  npm run dev
```
## Tech Stack

**Client:** REST-Client or Postman

**Server:** Node.JS, Express.JS
